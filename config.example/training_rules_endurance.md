# Training rules — endurance specialist

## Intensity control (hard brake)
- TSB < −20: immediately reduce to Z1 or rest day (mandatory, even if
  the plan says otherwise)

## Cardiac startup drift
- In the first ~10 minutes of a run, HR rises with a physiological lag
  (cardiac output lag)
- **Warmup in the intervals_icu format: do NOT set an HR-zone target**
  — otherwise the watch raises a false alarm even though pace and
  effort are correct
- HR orientation for the athlete may appear in the description text,
  but never as a zone target on the workout step
- For analysis: ignore the first 10 min of a run when evaluating HR

## Neuromuscular activation before sessions
- Before intervals / tempo: strides (3–4×) OR a short plyo sequence —
  both equivalent for running performance
- Prefer plyo when explosiveness / tendon work is already a training
  goal (cross-prep, ninja)
- Before flat Z2: strides are sufficient; plyo optional
- Don't include as a mandate — strides in the warmup achieve the same
  effect with less overhead

## Plyo as run warmup
- As a 5–10 min warmup block BEFORE a Z2 run: OK (same day, plyo first)
- Long run (>60 min) yesterday: only light plyo as a warmup
